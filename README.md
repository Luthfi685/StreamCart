# StreamCart 🚀

StreamCart adalah platform Live Streaming E-Commerce (Shoppertainment) modern yang menggabungkan fitur belanja interaktif dengan siaran langsung.

## Struktur Repository (Monorepo)

- 📁 **BACKEND**: Berisi sistem server utama menggunakan **Laravel**. Mengelola REST API, manajemen pengguna, transaksi, dan integrasi Agora.
- 📁 **FRONTEND**: Berisi aplikasi *mobile/web* interaktif menggunakan **Ionic/Angular**.
- 📁 **PYTHON_API**: Berisi *microservice* cerdas (Machine Learning) untuk sistem rekomendasi produk secara *real-time*.
